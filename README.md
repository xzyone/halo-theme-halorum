# Halorum

Halorum 是一款基于 Astro 构建的 Halo 主题，主打论坛式阅读体验。

## 功能
- 论坛式文章布局与评论体验
- 首页支持分页或瀑布流加载
- 支持 Markdown 主题与代码高亮主题切换
- 首页分类卡片与榜单可配置
- 支持自定义 CSS/JS 与 Head HTML

## 运行要求
- Halo >= 2.0.0
- Node.js >= 18.17.1（本地构建）

## 安装
1. 构建并打包主题（见下方）
2. 在 Halo 后台上传 zip 并启用主题

## 构建与打包
```bash
npm install
npm run build
npm run pack:theme
```
打包后会在项目根目录生成 `Halorum.zip`。

## 主题设置
主题提供以下分组：
- `basic`：Logo、联系信息、备案、Markdown/高亮主题
- `sidebar`（后台显示“首页设置”）：首页分页数量、加载方式、最新文章、分类卡片
- `forum`：评论排序与副标题
- `style`：主题强调色、圆角、顶部阴影
- `custom`：Head HTML、自定义 CSS/JS

## 项目结构
- `src/`：Astro 源码
- `templates/`：Halo 模板输出
- `public/`：静态资源

## English

Halorum is a Halo theme built with Astro and designed for a forum-style reading experience.

### Features
- Forum-style post layout and comment flow
- Home page supports pagination or infinite loading
- Markdown and code highlight themes selectable in settings
- Configurable home category cards and ranking sections
- Custom CSS/JS and head HTML injection

### Requirements
- Halo >= 2.0.0
- Node.js >= 18.17.1 (for local builds)

### Install
1. Build and package the theme (see below), then upload the generated zip in Halo.
2. Activate the theme in Halo admin.

### Build and Package
```bash
npm install
npm run build
npm run pack:theme
```
The packaged file will be generated as `Halorum.zip` in the project root.

### Theme Settings
The theme exposes settings under these groups:
- `basic`: logos, contact info, filing text, markdown and highlight themes
- `sidebar` (label: 首页设置): home page page size, load mode, latest posts, category cards
- `forum`: comment order and subtitle
- `style`: accent color, card radius, header shadow
- `custom`: head HTML, custom CSS, custom JavaScript

### Project Structure
- `src/`: Astro source pages and components
- `templates/`: Halo-ready templates generated by Astro
- `public/`: static assets bundled into the theme
